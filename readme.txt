Instrukcja właczenia aplikacji.
1. W folderze aplikacji znajdź plik jar nazwany KalkulatroWalut. Skopiuj jego ścieszkę razem z samym plikiem na końcu ścierzki.
2. Włącz terminal systemu windows.
3. Wpisz komendę: "java -Dfile.encoding=windows-1250 -jar 'ścierzka do pliku jar'". Wykonaj ją.
4. Program powinien już działać.
